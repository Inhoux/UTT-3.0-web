============

初始项目
----------

该页面首次使用于 REOS_SE_V3.0 项目
涉及 企业路由器，AC，AP产品

界面介绍
----------
不同于以往以 **功能模块** 为基础进行设置

本页面主要 以 **人** 和 **组织架构** 为基础进行设置,
比如：先导入一个公司的组织结构（部门，人际关系），后针对部门进行行为管控

页面开发
============

编码格式
----------
采用 **UTF-8**   格式 

数据通信格式
---------

支持 **html标准GET/POST** 和 **json** 两种格式
全新的页面 应该使用 **json** 格式，以增强兼容性
